# Technology Stack

## Backend
- **Language:** PHP 8.2+
- **Framework:** Laravel 12.0
- **Admin Panel:** Filament 4.0 (for IT Technicians and Admins)
- **Database:** MySQL
- **Architecture:** Monolithic Laravel application using MVC and Repository patterns where appropriate.

## Frontend (Public Portal)
- **Engine:** Blade Templates with Livewire (for dynamic, reactive ticket forms and tracking).
- **Styling:** Tailwind CSS 4.0
- **Build Tool:** Vite

## Testing & Quality
- **Test Framework:** Pest
- **Coding Standard:** Laravel Pint (standard Laravel style)

## Deployment & Environment
- **Web Server:** Nginx or Apache
- **Environment Management:** Laravel `.env`
